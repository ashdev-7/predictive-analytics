## Project Description

**TOPSIS** (Technique for Order of Preference by Similarity to Ideal Solution) is a multi-criteria decision-making (MCDM) method. It is used to rank alternatives by selecting the option that is closest to the ideal solution and farthest from the negative ideal solution.

This package provides a simple command-line tool to perform TOPSIS analysis on a CSV dataset.
